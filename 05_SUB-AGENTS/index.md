# 05_SUB-AGENTS Registry

This directory contains the configurations and protocols for specialized autonomous agents.

## Specialized Units
1. **Design_Stitch**: UI/UX Specialist (Google Stitch Standard).
2. **N8N_Grandmaster**: Backend & Automation orchestration.
3. **VPS_Sentry**: Server maintenance and deployment (Jules/Gemini CLI).
4. **Outreach_Prophet**: Cold outreach and traction management.

## Integration Protocol
Every sub-agent must have its own `.agent/rules/` entry and be triggered via the Manager (Gemini CLI).
