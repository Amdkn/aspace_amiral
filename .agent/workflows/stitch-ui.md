---
description: Transforme une interface ou un composant vers le standard Google Stitch.
---

1. Analyser le composant cible pour identifier les zones de Glassmorphism.
2. Appliquer les tokens de design définis dans `skill-stitch-design`.
3. Remplacer les arrondis par `rounded-[40px]`.
4. Injecter les Mesh Gradients via `index.css` ou styles inline.
5. Vérifier la cohérence visuelle avec le screenshot de référence `stitch_landing_page`.
