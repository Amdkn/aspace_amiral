# WORKFLOW: CREATE PRD

## Objectif
Définir le périmètre EXACT avant de coder. (Éviter le "Scope Creep").

## Template du PRD (À générer)
1. **North Star** : Quelle est la mission unique de cette fonctionnalité ?
2. **User Story** : "En tant que [Role], je veux [Action] pour [Bénéfice]".
3. **In Scope / Out of Scope** : Ce qu'on fait vs ce qu'on ne fait PAS.
4. **Tech Stack** : Outils précis (Supabase, N8N, React Native).
5. **Architecture** : Diagramme textuel des flux de données.

## Règle d'Or
Une fois le PRD validé : **CONTEXT RESET**.
L'agent doit oublier la conversation de brainstorming et ouvrir une nouvelle session propre avec le PRD comme seule vérité.
