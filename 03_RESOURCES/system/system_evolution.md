# SYSTEM EVOLUTION RITUAL

*Le concept :* Ne jamais juste corriger un bug. Corriger le *système* qui a permis le bug.

## Routine de fin de sprint / Session de débug
Après chaque correction majeure, lance ce prompt :

> "Amadeus, nous venons de corriger un bug sur [X]. Ne regarde pas le code. Regarde nos `global_rules.md` et nos `reference/`. Quelle règle manquait pour empêcher ce bug d'arriver ? Mets à jour le fichier de règles correspondant maintenant."
